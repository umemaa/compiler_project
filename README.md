# Mini Compiler Project

A Python-based Mini Compiler with a GUI built using Tkinter.

## Features
- Lexical Analysis
- Syntax Analysis
- Semantic Analysis
- Intermediate Code Generation
- Symbol Table
- Error Detection

## Technologies Used
- Python
- Tkinter
- Regular Expressions (Regex)

## Project Structure

- gui.py
- lexer.py
- parser.py
- semantic.py
- icg.py

## Run

```bash
python3 gui.py
```

## Sample Input

```text
int a = 10
int b = 20
a = b
```
